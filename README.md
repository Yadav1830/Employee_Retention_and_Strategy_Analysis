# Employee Retention Strategy Analysis

## Project Overview
This project analyzes employee data to understand the factors affecting employee retention and attrition. 
The goal is to identify key factors that influence employees leaving the organization and provide insights 
through interactive visualizations.

## Project Components

### 1. Streamlit Dashboard
An interactive HR analytics dashboard built using Streamlit that allows users to explore employee attrition trends and predict employee retention.

Main file:
app.py

### 2. Jupyter Notebook Analysis
The notebook contains the complete data analysis process including:
- Data preprocessing
- Exploratory Data Analysis (EDA)
- Business question analysis
- Visualizations

File:
Employee_Retention_Analysis.ipynb

### 3. Power BI Dashboard
A professional dashboard created in Power BI to visualize key insights such as:
- Attrition rate
- Department-wise attrition
- Age group analysis
- Job satisfaction trends

File:
Employee_Retention_Analysis_Dashboard.pbix

### 4. Dataset
The dataset used for analysis contains employee information including age, department, salary, overtime, satisfaction level, and more.

File:
cleaned_hr_data.csv

## Technologies Used

- Python
- Pandas
- NumPy
- Plotly
- Streamlit
- Power BI

# Employee Retention Strategy Analysis

## Project Overview
This project analyzes employee data to understand the factors affecting employee retention and attrition. It provides an interactive dashboard built using Streamlit for data exploration and prediction.

## Technologies Used
- Python
- Pandas
- NumPy
- Plotly
- Streamlit
- Scikit-learn

## Features
- Employee attrition analysis
- Interactive visualizations
- Key factors affecting employee retention
- Attrition prediction model

## Run Locally
pip install -r requirements.txt  
streamlit run app.py
